80 Remove Dup From Sorted Array II

LC 80 Remove Dup From Sorted Array II Given a sorted array nums, remove the duplicates in-place such that duplicates appeared at most twice and return the new length. Do not allocate extra space for another array, you must do this by modifying the input array in-place with O(1) extra memory. Given nums = [0,0,1,1,1,1,2,3,3], output [0,0,1,1,2,3,3] Solution 2020-09-04 Notes: Same thing as LC 26. class Solution: def removeDuplicates(self, nums: List[int]) -> int: if not nums: return 0 count = dict() count[nums[0]] = 1 slow, fast = 0, 1 while fast < len(nums): count[nums[fast]] = count.
