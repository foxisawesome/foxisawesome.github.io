Backtracking

Template:

def bt(path, choice_list):
    if (stopping condition):
        output.append(path[:])
        return 

    for i in choice_list:
        if not isValid():
            continue # isvalid is to prune the tree.

        path.append()
        bt(path, (choice_list - path))
        path.pop()

Note:

  • Time Complexity O(N!)
  • in output.append(path[:]), need deep copy. hence path[:].
