136 Single Number

LC 136 Single Number

Given a non-empty array of integers, every element appears twice except for one. Find that single one.

Note: Your algorithm should have a linear runtime complexity. Could you implement it without using extra memory?

Solution

2020-09-05 Notes:

  • Space O(1) solution is a bit tricky.
class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        stack = set()
        for num in nums:
            if num not in stack: stack.add(num)
            else: stack.remove(num)
        return stack.pop()

# Time O(N)
# Space O(N)

If we take XOR of zero and some bit, it will return that bit: $a \oplus 0 = a$ If we take XOR of two same bits, it will return 0: $a \oplus a = 0$

class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        a = 0
        for num in nums:
            a ^= num
        return a

class Solution(object):
    def singleNumber(self, nums):
        return 2 * sum(set(nums)) - sum(nums)
