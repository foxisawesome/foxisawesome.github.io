LC 977 Squares of a Sorted Array

Given an array of integers A sorted in non-decreasing order, return an array of the squares of each number, also in sorted non-decreasing order.

Input: [-4,-1,0,3,10]
Output: [0,1,9,16,100]

Solution

2020-09-28 Notes:

  • 先找到零的位置再向两边走。
class Solution:
    def sortedSquares(self, A: List[int]) -> List[int]:
        p, n = 0, len(A)
        while p < n and A[p] < 0:
            p += 1
        
        ans = []
        l, r = p-1, p
        while l>=0 and r<n:
            if abs(A[l]) >= abs(A[r]):
                ans.append(A[r]**2)
                r += 1
            else:
                ans.append(A[l]**2)
                l -= 1
        r = A[r:] if l<0 else A[:l+1][::-1]
        [ans.append(i**2) for i in r]
        return ans

# Time O(N)
# Space O(N)

one liner

class Solution:
    def sortedSquares(self, A: List[int]) -> List[int]:
        return sorted(x**2 for x in A)
