1 Two Sum

LC 1 Two Sum Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target. You may assume that each input would have *exactly* one solution, and you may not use the same element twice. Solution 2020-02-01 Notes: use hash table, one-pass. class Solution: def twoSum(self, nums: List[int], target: int) -> List[int]: if not nums or target is None: return [] d = dict() for i, val in enumerate(nums): if target - val not in d: d[val] = i else: return [d[target - val], i] # Time O(N) # Space O(N)
